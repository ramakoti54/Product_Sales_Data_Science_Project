***** Rectified Linear Unit(Relu) ****** Rectified Linear Unit (Relu) is a Supervised Learning method to perform either regression to understand the relationship among the input classifiers to predict the future class label values depending on the regression of training class label values or classification of class labels from the training classifier to predict the test label values and associate with one of the predictable labels.

The Relu process of an Machine Learning algorithm aids the ML Model in identifying the exponential inverse relation of the classifier class label values to perform prediction on the training class labels using the following function. n F(x) = sigma(1/(1+(e^-(1/1+x))) i = 1 Cost function for the output values are demonstrated through the following prediction entropy function.

  E(x,i,n,Y) = F(x) - Yi
Perform Prediction operation on the Cost function of Relu.

Steps to perform Algorithm in operating Rectified Linear Unit Supervised Learning method.

Step1: Acquainte Data from Csv or table structure. Step2: Clean Data after performing analysis on the data using python Transform data using feature engineering principles. Step3: Divide dataset into train and test data samples. and apply Relu on training dataset to perform inference from the dataset.

Step4: Optimize the model using Gradient descent and other optimization techniques and simulate the training process to resolve it to low cost function value.

Step5: Implement the model to predict results in accordance to test dataset from the trained model and compare data with test dataset.

Step6: Find confusion matrix values and entropy function to introduce backpropagation error in the function and repeat step 2 to 5 until there is an entropy to satisfy the minimum requirements for acceptance criteria.

Step7: Plot the output using respective plot diagram to illustrate results.
